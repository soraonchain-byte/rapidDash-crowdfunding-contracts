# RapidDash Crowdfunding Contracts 🚀

A secure Solana crowdfunding program built with Anchor.

## Features
- **Secure Vaults**: Uses PDA (Program Derived Addresses) to hold funds.
- **Individual Tracking**: Donor contributions are tracked via unique PDAs to prevent drain attacks.
- **Fixed Security**: Implemented individual refund logic to resolve "Phantom Refund" vulnerability.
- **Safety**: Built-in overflow protection and strict deadline validation.

## Tech Stack
- Framework: Anchor
- Language: Rust
- Target: Solana Devnet
